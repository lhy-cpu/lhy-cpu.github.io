学习历程与技术路线

1. 应用基石 (Java生态)

旅程始于可靠的 Java。通过构建跨平台的 Java 应用程序、开发 JavaWeb 后端服务、编写 Android 移动端应用，甚至为 Minecraft (Java版) 编写高自由度插件，确立了扎实的面向对象编程思维。

2. 深入底层 (C/C++ & 嵌入式硬件)

为了探究“计算机底层是如何工作的”，深入学习了 C/C++，并结合电子硬件开发积累了丰富的经验。在底层编程上，操作过 STC 宏晶单片机、STM32、ESP 系列到 Infineon AURIX™ TC 系列等芯片，并涉猎了树莓派 (Raspberry Pi)、NVIDIA Jetson、RK (Rockchip) 系列等各类开发板与嵌入式 Linux 应用开发。在硬件设计上，拥有自行绘制 PCB 的经验（包括中高速 USB2.0 设计与 6层 PCB+BGA 设计），独立设计过各类单片机应用板、DC-DC、飞控、电调、FOC控制板及开发板，并且目前正在积极探索架构更底层的 FPGA 技术。

3. 全栈拓展 (Web生态)

旨在掌控现代产品完整的生命周期，掌握了 HTML/CSS/JavaScript。这包括构建流畅的 Web 前端页面体验、开发异步 Node.js 后端，以及编写实用的浏览器插件。

4. 智能与自动化 (Python & AI)

目前正全身心探索 Python 前沿。从编写高效的小工具和脚本起步，如今的重心已深度聚焦于人工智能领域，尤其是大语言模型 (LLM)、AI 视觉以及计算机视觉 (CV)。
