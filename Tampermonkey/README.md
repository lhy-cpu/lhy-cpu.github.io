# Tampermonkey Scripts

这是一个 Tampermonkey 脚本集合，提供了一些便利的网页自动化和增强功能。

## 脚本列表

### 1. AutoDo
**文件**: `AutoDo.user.js` | (~~`AutoDo.js`~~)(Deprecated)

自动化脚本框架，可以对特定网站执行以下操作:
- **自动清除**: 移除页面上的特定元素
- **自动删除**: 删除烦人的弹窗、登录窗口等
- **自动点击**: 自动点击按钮或链接
- **自动添加样式**: 修改页面 CSS
- **自动添加属性**: 修改元素属性

支持全局规则 (`ALL`) 和网站特定规则，易于配置和扩展。

**适用场景**: 移除广告、解除复制限制、自动处理登录框等

---

### 2. AutoDo Mobile Edition
**文件**: `AutoDo-mobile.user.js` | (~~`AutoDo-mobile.js`~~)(Deprecated)

AutoDo 的移动设备优化版本，提供相同的自动化功能，针对移动网页进行了优化。

---

### 3. AutoGetMedia
**文件**: `AutoGetMedia.user.js`

强大的页面媒体(视频/音频)资源嗅探与下载工具。像浏览器开发者工具的网络面板一样工作。
- **链接拦截与显示**：拦截 Fetch API、XMLHttpRequest 发起的各类音视频文件（`mp4`, `mp3`, `ts`, `m3u8` 等），并在页面右上角悬浮展示。
- **防盗链绕过**：利用 Tampermonkey 高级特权发包，绕过各大视频网站的反跨域机制。
- **M3U8 深层解析**：完整支持 HLS (m3u8) 视频流解析，能支持多并发（上限5线程）下载其中的 TS 视频切片分块，并在本地缓存中自动合并为一个 `ts` 视频文件导出。
- **强力阻断器**：提供强制拦截源站媒体连接选项，用于面对播放器 "阅后即焚" 式的视频链接，保障其可以稳定无误地被独立下载接管。

## 安装方法

1. 安装 [Tampermonkey 浏览器扩展](https://www.tampermonkey.net/)
2. 访问脚本 [URL合集](https://lhy-cpu.github.io/Tampermonkey/) 或直接复制脚本内容
3. Tampermonkey 会提示您安装脚本
4. 确认安装后即可自动运行

## 使用前须知

- 脚本会在所有网站上运行，建议根据需要配置特定的网站规则
- 某些网站可能不兼容，可通过修改配置来适配
- 做备份后再修改脚本代码

## 更新

所有脚本都支持自动更新功能，Tampermonkey 会定期检查新版本。
